(Windows)使用微軟官方工具，解決軟體無法安裝或移除問題

使用者不小心刪除了不該刪除的系統暫存檔，造成某些程式，用控制台的解除安裝或變更程式無法正常移除軟體，想重新安裝覆蓋也無法成功。微軟官方有提供一個 「疑難排解員」的免費工具，可以快速修復這問題，功能如下：

案例：嘗試移除軟體失敗

要解除安裝軟體時，出現以下錯誤，以致於無法正常解除安裝。

使用微軟官方「疑難排解員」工具，解決無法安裝或移除問題

先至「疑難排解員」網頁，點擊「Download troubleshooter」下載。

此工具無需安裝，下載完成後，直接執行「MicrosoftProgram_Install_and_Uninstall.meta.diagcab」這個檔案即可執行，按「下一步」進行偵測。

偵測後，會詢問是你是「正在安裝」還是「正在解除安裝」時發生問題，因為我是要移除軟體，所以選擇「正在解除安裝」。(「正在安裝」操作上大致相同 )

選擇後將開始掃已安裝的應用程式

選擇你無法正常解除安裝的程式，按「下一步」繼續，若沒看到要解除安裝的程式，選「未列出」。

選擇「是，我要嘗解除安裝」

等待工具解決問題

出現「已修正」，代表問題已修復完成。

補充

此案例雖然顯示已修復，在「控制台」 → 「程式和功能」，程式清單中確實已看不到此應用程式，桌面捷徑也無法再執行，但實際上所有檔案在還在，後來是自行從檔案總管將所有檔案刪除乾淨。
