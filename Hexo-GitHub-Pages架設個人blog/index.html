Hexo

Hexo 是一個基於 Node.js 開發的網誌框架，具有下列幾項特點：

  • 使用Markdown語法
  • 一鍵部署，只需一個指令就能部署到GitHub Pages或Heroku等支援靜態網頁的空間
  • 豐富的外掛套件，可安裝外掛支援Jade, CoffeeScript
  • 輕量級，編輯速度快
  • 支援多樣Theme
